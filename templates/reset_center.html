🌐 Reset Internet Uptime Stats

Resets Internet uptime, today's Internet outages, longest Internet outage, last Internet outage, total outages, total recoveries, and accumulated downtime.

📈 Reset Internet Availability Reporting

Resets Internet availability reporting back to a clean baseline. This clears Internet uptime stats and Internet outage history together so availability starts fresh from the current time.

📜 Clear Event Log

Clears the Recent Event Log file. This removes the displayed system events from the log history.

🔌 Clear Cisco Events

Clears the Cisco syslog events file. This removes the current Cisco event history from the dashboard.

⚠ Clear Resolved Alerts

Clears resolved alert history only. Active alerts will remain in place and will not be removed.

📊 Reset Total Alert Counters

Resets the dashboard Total Alerts and Total Recoveries counters back to zero. This does not remove active alerts or alert history.

🚨 Reset All Alert History

Clears both active and resolved alert history. Current problems may reappear on the next scan if the device or link is still down.

📚 Reset Knowledge Base Notes

Deletes all Knowledge Base notes. This is useful only when you want to clear test notes or start over.

🔊 Reset Voice Alert Settings

Clears browser voice alert settings stored in this browser. This resets Voice Alerts back to OFF until you enable them again.

📲 Reset Notification Settings

Resets SMS/email notification settings to safe defaults and disables SMS alerts. You will need to re-enter notification details if you use this.

🌐 Clear Internet Uptime History

Clears the detailed Internet outage history log. This does not reset the current Internet uptime cards unless you also use Reset Internet Uptime Stats.
