🛡 Phase 12C – One-Click Restore Center

Restore Center allows On Watch to recover from a previous full backup directly from the web interface. Before any restore is performed, On Watch automatically creates an emergency backup of the current system.

1. Select Backup Choose a restore point
2. Confirm Restore Type RESTORE to continue
3. Emergency Backup Current system is protected first
4. Restart On Watch Service restarts after restore

Create Full Monitor Backup

Backup filename format: network-monitor-YYYYMMDD-phase12c-restore-ready.tar.gz

⚠ Restore Safety Notice

Restoring a backup replaces the current On Watch project files with the selected backup contents. The restore process should only be used when you intentionally want to return to a previous known-good version.

A pre-restore emergency backup is created automatically before the restore runs.
